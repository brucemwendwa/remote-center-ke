# Remote Center Kenya — Frontend

Premium e-commerce storefront for Remote Center Kenya (TV remotes, covers, smart accessories).

## Stack
React 18 · Vite · Tailwind CSS v3 · framer-motion · React Router v6 · Axios · @tanstack/react-query · Zustand · React Hook Form · Zod · react-hot-toast · lucide-react · recharts · socket.io-client.

## Run
```
cp .env.example .env
npm install
npm run dev
```

Build: `npm run build` — Preview: `npm run preview`.

Backend expected at `http://localhost:5000` with API `/api/v1` and Socket.io at root.
