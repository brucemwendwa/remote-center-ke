# Remote Center Kenya — Backend

E-commerce API for Remote Center Kenya (TV remotes, covers, accessories).

## Quick start

```bash
cp .env.example .env
# edit .env (set MONGO_URI etc.)
npm install
npm run seed   # optional — seeds 40 products + admin user
npm run dev
```

API base: `http://localhost:4000/api/v1`

Default seeded users:
- admin: `admin@remotecenter.ke` / `Admin@123`
- customer: `customer@remotecenter.ke` / `Customer@123`

## Stack
Node 18+ / Express / MongoDB (Mongoose) / JWT / Socket.IO / Stripe / M-Pesa Daraja / Cloudinary / Nodemailer.
